教室沿革

昭和56年

初代・前田昌純 教授 着任

昭和58年

「外科学講座第二外科学」として開講（現在の呼吸器・乳腺内分泌外科学）

平成11年3月

前田教授 退官

平成11年4月

第二代・横見瀬裕保 教授 就任。呼吸器外科、乳腺外科、内分泌外科への完全臓器別診療体制へ改革を推進

平成18年4月

講座名を「呼吸器・乳腺内分泌外科学講座」に変更。神戸市・神鋼病院に呼吸器外科を開設

平成20年4月

高知医療センターへ派遣開始

平成23年4月

高松市立みんなの病院に呼吸器外科を開設

平成27年10月

坂出市立病院に開設

平成28年4月

西宮市・明和病院に開設

令和4年3月

横見瀬教授 退官

令和4年4月

第三代・矢島俊樹 教授 就任

香川大学医学部附属病院 呼吸器・乳腺内分泌外科
